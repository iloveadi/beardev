개인정보처리방침

최종 업데이트: 2026년 2월 26일  |  로또 확률 예측기 (Lotto Predictor)

1. 개요

본 개인정보처리방침은 크롬 확장 프로그램 로또 확률 예측기 (Lotto Predictor) (이하 "앱")의 개인정보 처리 방식을 설명합니다.

✅ 본 앱은 사용자의 개인 식별 정보를 일절 수집, 저장, 전송하지 않습니다.

2. 수집하는 정보

본 앱이 다루는 데이터는 다음과 같습니다. 이는 모두 사용자의 기기 내에만 저장되며 외부로 전송되지 않습니다.

  • 통계 가중치 설정값: 사용자가 조절한 슬라이더 수치 (0~100). 앱 환경설정 유지 목적으로 기기 로컬 스토리지에 저장됩니다.
  • 로또 당첨 이력 데이터: 동행복권 공개 API에서 수신한 역대 1등 당첨 번호. 통계 분석에 사용되며 기기 로컬 스토리지에 저장됩니다.

3. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 결제 또는 금융 정보
  • 브라우징 기록
  • 사용자 생성 콘텐츠

4. 외부 API 사용

앱은 최신 로또 당첨 번호를 가져오기 위해 다음 외부 서비스를 이용합니다:

  • 동행복권 (dhlottery.co.kr) — 공식 당첨 번호 API
  • AllOrigins API — CORS 우회 프록시

이 요청에는 어떠한 사용자 식별 정보도 포함되지 않습니다.

5. 데이터 저장 및 보안

모든 데이터는 chrome.storage.local을 통해 사용자의 기기 내에만 저장됩니다. 개발자 또는 제3자 서버로 전송되지 않습니다.

6. 제3자 공유

본 앱은 어떠한 데이터도 제3자와 공유하지 않습니다.

7. 아동 개인정보 보호

본 앱은 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 면책 조항

⚠️ 본 앱이 제공하는 번호는 통계적 알고리즘에 기반한 추천일 뿐, 실제 당첨을 보장하지 않으며 결과에 대한 책임은 사용자에게 있습니다.

9. 방침 변경

본 방침은 변경될 수 있으며, 변경 시 본 페이지를 통해 공지됩니다. 지속적인 앱 사용은 변경된 방침에 대한 동의로 간주됩니다.
